# Fawang Notebook Android

This project packages the notebook as an offline Android WebView app.

Build locally with Android Studio:

1. Open the `android-app` folder in Android Studio.
2. Let Android Studio sync Gradle.
3. Build `app` as a debug APK.

Build on GitHub:

The workflow at `.github/workflows/build-android-apk.yml` builds a debug APK and uploads it as an artifact.
