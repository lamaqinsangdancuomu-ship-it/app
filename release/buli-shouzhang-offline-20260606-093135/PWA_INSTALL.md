# PWA 安装说明

这个文件夹已经整理成 PWA 安装版。发布时请把 `C:\Users\wangb\Documents\app` 整个文件夹上传到支持 HTTPS 的网站空间，例如 GitHub Pages、Netlify、Vercel 或自己的 HTTPS 域名。

## 安卓安装

1. 用 Chrome 打开发布后的 HTTPS 地址。
2. 点击页面里的“安装到手机”，或点右上角菜单。
3. 选择“安装应用”或“添加到主屏幕”。

## iPhone 安装

1. 用 Safari 打开发布后的 HTTPS 地址。
2. 点击底部分享按钮。
3. 选择“添加到主屏幕”。

## 注意

- `file://` 直接打开不能安装为 PWA。
- 手机和电脑同 Wi-Fi 的 `http://局域网IP:端口` 适合预览，但正式安装推荐使用 HTTPS 地址。
- 安装后笔记内容会保存在手机浏览器本地数据中，导出笔记可以备份 JSON 文件。
